

🔴 Red Light, Green Light (Pygame Edition) 🟢


A 2D survival game inspired by the "Red Light, Green Light" challenge from Squid Game. Built entirely in Python using the Pygame library, this project features randomized game logic, a state-machine architecture, and 100% programmatically generated graphics.




🎮 How to Play
Objective: Reach the black finish line at the top of the screen before the timer hits zero.

Controls: Use the Arrow Keys to move your character.

The Rules: * Green Light: You are free to move.

Red Light: STOP immediately. If the game detects any movement (key presses) during this phase, you are eliminated.

Restart: Press SPACE on the Game Over screen to try again.
